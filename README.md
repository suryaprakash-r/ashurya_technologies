## 📌 Overview

Ashurya Technologies is a modern software company website built using Django.

The platform showcases:

- Company Services
- Portfolio Projects
- Blog Articles
- Testimonials
- Contact & Lead Collection
- Internal Admin Dashboard
- Project Management System
- Lead Tracking System

The goal of this platform is to provide businesses with a professional digital presence while also giving administrators powerful tools to manage clients, projects, blogs, and inquiries.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/suryaprakash-r/ashurya_technologies.git
```

### Move into Project

```bash
cd ashurya_technologies
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Run Server

```bash
python manage.py runserver
```
